REST(Representational State Transfer)는 월드 와이드 웹(WWW : World Wide Web)과 같은 분산 하이퍼미디어 시스템을 위한 소프트웨어 아키텍처의 한 형식

쉽게 번역하자면 [네트워크에서 통신을 구성할 때 이런 구조로 설계하라는 지침]이라고 한다.

Representational : 대표성  
State : 상태  
Transfer : 옮기다
